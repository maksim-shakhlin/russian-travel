# Путешествия по России
### Описание
Одностраничный сайт, разработанный в рамках обучения в Яндекс Практикуме на курсе «Веб-разработчик». На нем приведена подборка фотографий и заметок, призванных пробудить интерес к внутреннему туризму.
### Технологии
Сайт написан на чистом html и css по методологии БЭМ.
